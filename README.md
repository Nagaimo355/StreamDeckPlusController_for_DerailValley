# StreamDeckPlusController_for_DerailValley
Elgato製 Stream Deck Plusを用いたDerail Valley用コントローラー。SL以外のディーゼル/バッテリー機関車で使用可能です。

# 必要なもの
- Stream Deck Plus
- Stream Deck ソフトウェア
- AutoHotkey（v2.0）

# 導入方法
１．「StreamDeck_SupportFile.zip」をダウンロード＆解凍し、フォルダ「StreamDeck_SupportFile」をデスクトップに配置します。ZIP解凍する際に最上位フォルダ「StreamDeck_SupportFile」がひとつ余分に作成されますが、内側のフォルダ「StreamDeck_SupportFile」で大丈夫です。

２．フォルダ「Derail Valley」の「260317_StreamDeckPlus_DerailValley.streamDeckProfilesBackup」をダブルクリックし、バックアップファイルを適用します。

３．AutoHotkey（v2.0）をダウンロード・インストールします。
これによってキーボード自動操作スクリプトが動作可能になります。

ダウンロード：https://www.autohotkey.com/

４．ゲーム「Derail Valley」のキーバインド設定をAHKスクリプトと同じにします。もしくは、ご自身のキーバインド設定に合わせてAHKスクリプトを変更します。参考までに私のキーバインド設定をZIPファイル「ControlBind_ScreenShot.zip」にまとめています。どちらにせよ、かなりの苦行になるかと思いますが頑張って下さい。
